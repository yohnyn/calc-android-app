## 本轮完成项

- 仓位参数顶部币种显示仅保留图标 + 缩写，去掉全称显示。
- 币种选择和结算模式改为统一轻量选择卡，下拉箭头右侧居中。
- 做多/做空与全仓/逐仓选择已恢复到仓位参数内。

- 杠杆只保留单一轻量下拉，不再展示独立杠杆数值卡或一整排快捷 chip。

- `目标价格` 后增加 `强平价格` 选项，同区展示；展开内容贴在该选择行下方，不再作为页面独立模块。
- 所有直接使用的下拉菜单统一为白底、细边框、轻阴影和紧凑行距，避免默认菜单和触发控件割裂。
- 开仓价 / 平仓价中间箭头已下移居中。
- 止盈/止损、强平价格和目标价格点击时不再触发整块背景突兀变色。
- 首页底部恢复支持作者入口；历史 / 方案库 / 设置仍作为底部低权重入口，`重置` 不回到首页底部。
- 补仓助手入口恢复轻量白底卡，右侧 `展开` 文案和箭头居中。
- 补仓助手在没有主方案结果时，也可从方案库选择已保存方案填入当前持仓摘要。
- 方案库入口可直接从底部进入历史页的方案库标签。

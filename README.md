# calcApp

This workspace contains the Android scaffold and the product/development spec for a personal USDT futures profit calculator.

## Files

- `APP_DEVELOPMENT_SPEC.md`: Product structure, formulas, constraints, development steps, and acceptance checklist.
- `android-app/`: Minimal Android app scaffold.

## Open The Project

Open this folder in Android Studio:

```text
android-app
```

Install Android 16 SDK / API 36 if Android Studio prompts for it.

## Environment Needed

- JDK 17 or Android Studio bundled JDK.
- Android SDK API 36.
- Gradle or Android Studio Gradle sync.

This local shell currently does not include Gradle or Android SDK, so command-line build verification cannot be completed here.

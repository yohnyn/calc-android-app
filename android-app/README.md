# FuturesCalculator Android App

This is the base Android project scaffold for the USDT futures profit calculator.

Read the root-level `APP_DEVELOPMENT_SPEC.md` before implementing features.

## Current State

- Kotlin Android app scaffold.
- Jetpack Compose entry screen.
- Material 3 theme.
- Domain/model/viewmodel/ui/util package structure.
- Placeholder calculators and models matching the product spec.

## Build

Open `android-app/` in Android Studio, install Android 16 SDK / API 36 if prompted, then run the `app` configuration.

If using command line, generate or install a Gradle wrapper first, then run:

```sh
./gradlew :app:assembleDebug
```
